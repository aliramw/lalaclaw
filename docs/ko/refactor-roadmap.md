- 세션과 런타임 컨트롤러를 더 일관되게 정리하기
- 대기열 처리, 복원, 스트리밍을 위한 컨트롤러 수준 테스트 보강
- 문서와 현지화 워크플로 자동화 강화
- 복잡한 인스펙터 로직을 더 작은 UI / 데이터 모듈로 분리하기
