# 인스펙터, 파일 미리보기, 추적

오른쪽 인스펙터는 현재 세션의 실행 데이터를 한곳에 모아 보여줍니다.

## 주요 영역

- 실행 로그: 도구 호출, 입력/출력, 파일 변경
- 파일: 세션 파일과 workspace 파일의 분리된 트리
- 요약: 점프 링크가 있는 응답 요약
- 환경: 에이전트, 모델, 런타임, gateway 정보
- 협업: 하위 에이전트와 작업 관계
- 미리보기: 파일, 표, 미디어 미리보기

## Workspace 파일

- workspace 트리는 단계적으로 로드됩니다
- 폴더를 필터링, 새로고침, 다시 로드할 수 있습니다
- CSV 와 XLSX 파일은 미리보기 패널에서 바로 열 수 있습니다
