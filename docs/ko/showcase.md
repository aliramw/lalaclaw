# 제품 쇼케이스

LalaClaw 는 chat, 세션 제어, 관측 기능을 하나의 인터페이스에 결합합니다.

## 눈에 띄는 점

- 빠르게 전환할 수 있는 여러 agent 세션
- VS Code 스타일의 파일 트리, preview, context menu
- timeline, environment 상태, summary 를 위한 오른쪽 패널
- 다국어 UI, theme, 단축키 도움말
