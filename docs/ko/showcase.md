# 제품 소개

LalaClaw 는 채팅, 세션 제어, 관측 기능을 하나의 인터페이스에 결합합니다.

## 눈에 띄는 점

- 빠르게 전환할 수 있는 여러 에이전트 세션
- VS Code 스타일의 파일 트리, 미리보기, 컨텍스트 메뉴
- 타임라인, 환경 상태, 요약을 위한 오른쪽 패널
- 다국어 UI, 테마, 단축키 도움말
