# Keyboard Shortcuts

## Global

- `Cmd/Ctrl + /`: open shortcut help
- `Cmd/Ctrl + N`: start a new session
- `Cmd/Ctrl + 1-9`: switch tabs by index
- `Cmd/Ctrl + Left`: switch to the previous tab
- `Cmd/Ctrl + Right`: switch to the next tab

## Appearance

- `Shift + Cmd/Ctrl + F`: switch to system theme
- `Shift + Cmd/Ctrl + L`: switch to light theme
- `Shift + Cmd/Ctrl + D`: switch to dark theme

## Composer

- `Enter`: insert a newline
- `Shift + Enter`: send
- Double-tap `Enter`: send
- `ArrowUp`: previous prompt history entry
- `ArrowDown`: next prompt history entry

## Dialogs and Menus

- `Esc`: close the shortcut dialog, mention menu, or some temporary context menus

## Global Input Capture

Besides explicit shortcuts, the page has one small typing convenience:

- If focus is not in the composer and you type a regular character or space, the app moves focus into the composer and inserts that character for you

That means you often do not need to click the input before typing.
