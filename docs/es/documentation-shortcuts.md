# Atajos de teclado

- `Cmd/Ctrl + /`: abrir ayuda de atajos
- `Cmd/Ctrl + N`: nueva sesion
- `Cmd/Ctrl + 1-9`: cambiar de pestana por numero
- `Cmd/Ctrl + Left`: pestana anterior
- `Cmd/Ctrl + Right`: pestana siguiente
- `Shift + Cmd/Ctrl + F`: tema del sistema
- `Shift + Cmd/Ctrl + L`: tema claro
- `Shift + Cmd/Ctrl + D`: tema oscuro
- `Enter`: nueva linea
- `Shift + Enter`: enviar
