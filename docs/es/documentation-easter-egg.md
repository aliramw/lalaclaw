# Huevo de pascua

El icono `🦞` en la zona de marca superior izquierda es clicable.

Al pulsarlo:

- La langosta inicia una animacion por la pagina
- El icono fijo se oculta temporalmente
- Al terminar, el icono reaparece

Solo se ejecuta una animacion a la vez y no bloquea la interaccion normal.
