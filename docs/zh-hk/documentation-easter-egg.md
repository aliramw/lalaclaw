# 彩蛋

LalaClaw 在左上角龍蝦品牌元素上加入了一個小彩蛋。

- 點擊後可能會觸發裝飾性動畫
- 不會影響 chat 或 runtime 行為
- 動畫仍會跟隨目前 theme 與 UI 設定
