# 鍵盤快捷鍵

- Cmd/Ctrl + /：開啟快捷鍵說明
- Cmd/Ctrl + N：新工作階段
- Cmd/Ctrl + 1-9：用數字切換工作階段
- Cmd/Ctrl + Left：上一個分頁
- Cmd/Ctrl + Right：下一個分頁
- Shift + Cmd/Ctrl + F：系統主題
- Shift + Cmd/Ctrl + L：淺色主題
- Shift + Cmd/Ctrl + D：深色主題

Enter 傳送模式：

- Enter：傳送
- Shift + Enter：換行

Double Enter 傳送模式：

- 連按兩次 Enter：傳送
- Shift + Enter：傳送
- Enter：換行
- Cmd + Shift + . / Ctrl + Shift + .：開始或停止語音輸入
