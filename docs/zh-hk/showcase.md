# 產品展示

LalaClaw 把 chat、工作階段控制與觀察能力整合到同一個介面。

## 特色

- 可快速切換的多個 agent 工作階段
- VS Code 風格的檔案樹、preview 與 context menu
- 用於 timeline、environment 狀態與 summaries 的右側面板
- 多語言介面、主題與快捷鍵說明
