# 介面總覽

LalaClaw 的主畫面可分為三個部分：上方工作階段控制列、中間聊天工作區，以及右側檢查器。

## 頂部控制區

- 頂部可控制 Agent、模型、上下文、隊列、主題與語言
- 思考模式支援 off / minimal / low / medium / high / xhigh / adaptive
- 語言切換支援 中文 / 繁體中文（香港） / English / 日本語 / 한국어 / Français / Español / Português / Deutsch / Bahasa Melayu / தமிழ்
- 快捷鍵說明與新工作階段操作會常駐顯示

## 聊天工作區

- 每個 Agent 各有一個工作階段標籤
- 對話區包含串流回覆與附件預覽
- 輸入框支援文字、@ 提及、檔案上傳與停止操作

## 右側檢查器

- 執行記錄
- 檔案
- 摘要
- 環境
- 協作
- 預覽
