# 快捷键说明

## 全局

- `Cmd/Ctrl + /`：打开快捷键帮助
- `Cmd/Ctrl + N`：开启新会话
- `Cmd/Ctrl + 1-9`：按序号切换标签页
- `Cmd/Ctrl + Left`：切换到上一个标签页
- `Cmd/Ctrl + Right`：切换到下一个标签页

## 外观

- `Shift + Cmd/Ctrl + F`：切换为跟随系统主题
- `Shift + Cmd/Ctrl + L`：切换为浅色主题
- `Shift + Cmd/Ctrl + D`：切换为深色主题

## 输入框

- `Enter`：插入换行
- `Shift + Enter`：发送
- 连按两次 `Enter`：发送
- `ArrowUp`：上一条输入历史
- `ArrowDown`：下一条输入历史

## 弹窗与菜单

- `Esc`：关闭快捷键弹窗、提及菜单或部分临时菜单

## 全局输入接管

除了显式快捷键，页面还有一个输入便利行为：

- 如果焦点不在输入框里，你直接输入普通字符或空格，应用会把焦点切到输入框并插入该字符

这意味着很多时候你不需要先手动点一下输入框。
