# Gambaran Antara Muka

Skrin utama LalaClaw paling mudah difahami sebagai tiga bahagian: pengepala kawalan sesi, ruang kerja chat dan inspector di sebelah kanan.

## Pengepala dan kawalan sesi

- Kawalan agent, model, context, queue, tema dan bahasa berada di bahagian atas
- Mod pemikiran menyokong off, minimal, low, medium, high, xhigh dan adaptive
- Penukaran bahasa tersedia untuk 中文 / 繁體中文（香港） / English / 日本語 / 한국어 / Français / Español / Português / Deutsch / Bahasa Melayu / தமிழ்
- Dialog pintasan dan tindakan sesi baharu sentiasa boleh dicapai

## Ruang kerja chat

- Tab sesi mengikut agent
- Sejarah mesej dengan balasan streaming dan lampiran
- Composer dengan teks, @ mention, muat naik fail dan butang stop

## Inspector kanan

- Run Log
- Files
- Summaries
- Environment
- Collab
- Preview
