# Showcase Produk

LalaClaw menggabungkan chat, kawalan sesi dan pemerhatian dalam satu antara muka.

## Perkara yang menonjol

- Banyak sesi agent dengan pertukaran pantas
- Pokok fail ala VS Code dengan pratonton dan menu konteks
- Panel kanan untuk timeline, keadaan persekitaran dan ringkasan
- Antara muka berbilang bahasa, tema dan bantuan pintasan
