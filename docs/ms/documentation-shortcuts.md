# Pintasan Papan Kekunci

- Cmd/Ctrl + /: buka bantuan pintasan
- Cmd/Ctrl + N: sesi baharu
- Cmd/Ctrl + 1-9: tukar sesi mengikut nombor
- Cmd/Ctrl + Left: tab sebelumnya
- Cmd/Ctrl + Right: tab seterusnya
- Shift + Cmd/Ctrl + F: tema sistem
- Shift + Cmd/Ctrl + L: tema terang
- Shift + Cmd/Ctrl + D: tema gelap

Mod Enter untuk hantar:

- Enter: hantar
- Shift + Enter: baris baharu

Mod Double Enter untuk hantar:

- Enter dua kali: hantar
- Shift + Enter: hantar
- Enter: baris baharu
