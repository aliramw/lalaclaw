## 阶段 3：性能优化（第 5-6 天）

### 3.1 React 优化

**优化点**：
- 大组件用 `React.memo`
- 回调用 `useCallback`
- 计算用 `useMemo`
- 避免匿名函数作为 props

### 3.2 消息列表虚拟滚动

**方案**：使用 `react-window` 或 `@tanstack/react-virtual`

**收益**：
- 1000+ 消息时性能提升 10x
- 内存占用减少 80%

### 3.3 代码分割

**方案**：
- 路由级懒加载
- Inspector 面板按需加载
- 文件预览组件懒加载

### 验收标准

- [ ] 首屏加载 < 2s
- [ ] 1000 消息滚动流畅 (60fps)
- [ ] 内存占用 < 200MB
