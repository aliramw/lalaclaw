# அரட்டை, இணைப்புகள் மற்றும் கட்டளைகள்

## செய்தி அனுப்புதல்

- Enter அழுத்தி அனுப்பு முறை: Enter அனுப்பும்; Shift + Enter புதிய வரி சேர்க்கும்
- Double Enter அழுத்தி அனுப்பு முறை: இரண்டு முறை Enter அனுப்பும்; Shift + Enter கூட அனுப்பும்; ஒரு முறை Enter புதிய வரி சேர்க்கும்
- ArrowUp மற்றும் ArrowDown முன் பயன்படுத்திய prompt வரலாற்றைக் காட்டும்
- Stop நடப்பு பதிலை நிறுத்தும்

## வரிசை

தற்போதைய தாவல் பிஸியாக இருந்தால், புதிய செய்தி வரிசையில் சேர்க்கப்பட்டு, நடப்பு பதில் முடிந்ததும் தானாக அனுப்பப்படும்.

## இணைப்புகள் மற்றும் slash கட்டளைகள்

- படங்களுக்கு முன்னோட்டம் கிடைக்கும்
- உரைக் கோப்புகள் வாசிக்கப்பட்டு மிக நீளமானால் சுருக்கப்படும்
- /model, /think, /new, /reset போன்ற slash கட்டளைகள் ஆதரிக்கப்படுகின்றன
