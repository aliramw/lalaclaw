# இடைமுக அறிமுகம்

LalaClaw இன் முக்கிய திரை மூன்று பகுதிகளைக் கொண்டது: session header, chat workspace மற்றும் வலது புற inspector.

## Header மற்றும் session controls

- மேல் பகுதியில் agent, model, context, queue, theme மற்றும் language controls இருக்கும்
- சிந்தனை முறை off, minimal, low, medium, high, xhigh மற்றும் adaptive ஆகியவற்றை ஆதரிக்கிறது
- மொழி மாற்றம் 中文 / 繁體中文（香港） / English / 日本語 / 한국어 / Français / Español / Português / Deutsch / Bahasa Melayu / தமிழ் க்கு கிடைக்கும்
- shortcut dialog மற்றும் புதிய session action எப்போதும் அணுகக்கூடியவை

## Chat workspace

- ஒவ்வொரு agent க்கும் தனி session tabs
- streaming பதில்கள் மற்றும் attachments உடன் message history
- text, @ mention, file upload மற்றும் stop action உடன் composer
