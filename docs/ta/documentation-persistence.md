# உள்ளூர் நிலைபேர் மற்றும் மீட்பு

மீண்டும் ஏற்றிய பிறகு இடைமுகத்தை விரைவாக மீட்டமைக்க LalaClaw சில UI நிலைகளை உள்ளூரில் சேமிக்கிறது.

- திறந்த தாவல்கள் மற்றும் செயலில் உள்ள அமர்வு
- ஆய்விப் பலகையின் அகலம்
- அரட்டை எழுத்தளவு
- தேர்ந்தெடுத்த மொழி மற்றும் தோற்றம்

மீட்டமைக்கும் போது, runtime தரவுகளும் சேமிக்கப்பட்ட நிலையும் மீண்டும் ஒத்திசைக்கப்படுகின்றன; உரையாடல் அமைதியாக கைவிடப்படாது.
