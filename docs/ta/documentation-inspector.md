# Inspector, கோப்பு முன்னோட்டம் மற்றும் Tracing

வலது புற inspector தற்போதைய session-இற்கான execution data-ஐ ஒன்றாகக் காட்சிப்படுத்துகிறது.

## முக்கிய பகுதிகள்

- Run Log: tool calls, input/output மற்றும் file changes
- Files: தனி session tree மற்றும் workspace tree
- Summaries: jump links உடன் reply summaries
- Environment: agent, model, runtime மற்றும் gateway தகவல்கள்
- Collab: child agent மற்றும் task உறவுகள்
- Preview: files, spreadsheets மற்றும் media previews

## Workspace கோப்புகள்

- workspace tree படிப்படியாக ஏற்றப்படுகிறது
- folders-ஐ filter செய்யவும், refresh செய்யவும், context menu மூலம் மீண்டும் load செய்யவும் முடியும்
- CSV மற்றும் XLSX கோப்புகளை preview panel-ல் நேரடியாகத் திறக்கலாம்
