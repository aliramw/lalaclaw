# ஆய்விப் பலகம், கோப்பு முன்னோட்டம் மற்றும் தடமறிதல்

வலது புற ஆய்விப் பலகம் தற்போதைய அமர்விற்கான செயல்பாட்டு தரவுகளை ஒரே இடத்தில் காட்டுகிறது.

## முக்கிய பகுதிகள்

- இயக்கப் பதிவு: tool அழைப்புகள், உள்ளீடு/வெளியீடு மற்றும் கோப்பு மாற்றங்கள்
- கோப்புகள்: தனி அமர்வு மரம் மற்றும் workspace மரம்
- சுருக்கங்கள்: தாவும் இணைப்புகளுடன் பதில் சுருக்கங்கள்
- சூழல்: ஏஜென்ட், மாதிரி, runtime மற்றும் gateway தகவல்கள்
- ஒத்துழைப்பு: child agent மற்றும் task உறவுகள்
- முன்னோட்டம்: கோப்புகள், spreadsheet-கள் மற்றும் media முன்னோட்டங்கள்

## Workspace கோப்புகள்

- workspace மரம் படிப்படியாக ஏற்றப்படுகிறது
- கோப்புறைகளை filter செய்யவும், refresh செய்யவும், context menu மூலம் மீண்டும் load செய்யவும் முடியும்
- CSV மற்றும் XLSX கோப்புகளை முன்னோட்ட பலகையில் நேரடியாகத் திறக்கலாம்
