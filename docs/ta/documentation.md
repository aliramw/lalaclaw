## பரிந்துரைக்கப்படும் வாசிப்பு வரிசை

1. முதலில் [விரைவு தொடக்கம்](./documentation-quick-start.md) படிக்கவும்.
2. பிறகு [இடைமுக அறிமுகம்](./documentation-interface.md) மற்றும் [அரட்டை, இணைப்புகள் மற்றும் கட்டளைகள்](./documentation-chat.md) படிக்கவும்.
3. வலது பக்க பலகத்தைப் புரிந்துகொள்ள [ஆய்விப் பலகம், கோப்பு முன்னோட்டம் மற்றும் தடமறிதல்](./documentation-inspector.md) படிக்கவும்.
4. ஏஜென்ட்கள், மாதிரிகள், மற்றும் mock / openclaw வேறுபாடு குறித்து [அமர்வுகள், ஏஜென்ட்கள் மற்றும் இயக்க முறைகள்](./documentation-sessions.md) படிக்கவும்.
5. பயன்பாடு மற்றும் பிழைத்திருத்தத்திற்காக [விசைப்பலகை குறுக்கு வழிகள்](./documentation-shortcuts.md), [உள்ளூர் நிலைபேர் மற்றும் மீட்பு](./documentation-persistence.md), [API மற்றும் சிக்கல் தீர்வு](./documentation-api-troubleshooting.md) படிக்கவும்.
