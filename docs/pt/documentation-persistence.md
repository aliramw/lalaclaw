# Persistencia local e recuperacao

O frontend salva localmente:

- Aba ativa de chat e inspector
- Historico de mensagens
- Rascunhos
- Historico de prompts
- Tema e idioma
- Largura do inspector
- Tamanho da fonte
- Estado de scroll
- Turnos pendentes
