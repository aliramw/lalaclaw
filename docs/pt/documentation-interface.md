# Visao geral da interface

A tela principal do LalaClaw tem tres partes: controles de sessao no topo, area de chat e inspector na direita.

## Topo

- Troca de modelo
- Uso de contexto
- Fast mode
- Thinking mode
- Idioma
- Tema
- Ajuda de atalhos
- A lagosta clicavel do canto superior esquerdo

## Conversa

- Abas por agente
- Cabecalho com agente atual e estado
- Conversa com respostas em streaming
- Composer com texto, `@`, anexos e parada de resposta
