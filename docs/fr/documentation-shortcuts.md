# Raccourcis clavier

## Global

- `Cmd/Ctrl + /` : ouvrir l'aide des raccourcis
- `Cmd/Ctrl + N` : démarrer une nouvelle session
- `Cmd/Ctrl + 1-9` : changer d'onglet par numéro
- `Cmd/Ctrl + Left` : onglet précédent
- `Cmd/Ctrl + Right` : onglet suivant

## Apparence

- `Shift + Cmd/Ctrl + F` : thème système
- `Shift + Cmd/Ctrl + L` : thème clair
- `Shift + Cmd/Ctrl + D` : thème sombre

## Composer

- `Enter` : nouvelle ligne
- `Shift + Enter` : envoyer
- Double `Enter` : envoyer
- `ArrowUp` : historique précédent
- `ArrowDown` : historique suivant

## Menus et dialogues

- `Esc` : fermer le dialogue de raccourcis, le menu de mention ou certains menus contextuels
